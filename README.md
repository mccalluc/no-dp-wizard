# no-dp-wizard
Testbed to explore what kind of summary data curators can use

## Use
```
pip install polars
pip install git+https://github.com/mccalluc/no-dp-wizard
python -m no_dp_wizard your-own.tsv
```
We can make a cleaner process, if this is actually useful.

pip install -r requirements-dev.txt
